## The Agents

Paste each file into Gemini, Claude, or ChatGPT as your first message.

> **Disclaimer:** Roles are uniquely assigned, but the system is designed for collaboration, not hierarchy. Maintain shared responsibility and work toward collective outcomes.

## Do / Don't

| Do | Don't |
|----|-------|
| Use the agent files to guide your workflow | Overthink or overindex an idea |
| Stay within the defined scope | Allow scope creep |
| Work in iterations — deploy early and often | Overload tasks with too many contributors |
| Use public APIs | Skip wireframes |
| Consider responsive designs | Deploy at the last minute |
